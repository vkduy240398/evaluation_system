Báo cáo thay đổi UI — admin-user/user-list & user-detail

Ngày áp dụng: 2026-06-28  |  Rule cơ sở: UI-RULE.md (đã hiệu chỉnh theo list-user-evaluation & period-evaluation-detail)

Mục tiêu: Áp dụng rule UI thống nhất vào toàn bộ màn hình user-list, user-detail và các Modal liên quan. Sau thay đổi, tất cả màn hình sẽ đồng nhất về spacing, typography, button pattern, table locale, và Modal pattern với màn hình chuẩn.

  • 7 file đã được chỉnh sửa
  • Xóa các constant nội bộ dư thừa (CELL_STYLE, LABEL_STYLE) vì CSS global đã xử lý
  • Thay thế Popconfirm anti-pattern bằng ModalCustomComponent
  • Chuẩn hóa spacing về thang 4px (page) và 15px (modal)
  • Chuẩn hóa Modal result/notify: title dạng string thuần, footer canh phải
  • ROW_GUTTER trong Modal: [10,10] → [15,15]

G.2 Notify Modal footer — Root cause & fix

Root cause: .ant-modal-footer có CSS mặc định text-align: right.
<Space> render là display: inline-flex → phần tử inline → bị đẩy sang phải theo text-align của parent.
<div> là block-level → chiếm 100% chiều rộng → không bị text-align của parent ảnh hưởng → justify-content: flex-start hoạt động đúng.
FileTrước (sai — RIGHT)Sau (đúng — LEFT)
